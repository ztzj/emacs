1 emcas常用命令

C = ctrl, M = Alt,Commond

1.1 光标移动

C-v 向下翻页 M-v 向下翻页
C-b 向左（back） C-f 向右（forward）
C-n 向下（next） C-p 向上（previous）
M-b 上一个单词 M-f 下一个单词
C-a 行首 C-e 行尾
M-a 句首 M-e句尾
M-< 文件头 M-> 文件尾
M-g g 跳到某一行

1.2 选择区域

C-@ 标记

1.3 删除剪切复制粘贴

C-d 向后删除
C-k 删除光标后至行尾
M-w 复制区域
C-w 剪切/删除区域
C-y 粘贴
M-y 滚动选择粘贴内容

1.4 查找替换

C-s 向前查找
C-s C-r 向后查找
M-% 替换

1.5 文件操作

C-x C-f 打开文件（find/create）
C-x C-s 保存文件
C-x C-w 另存为
C-k 关闭文件

1.6 窗口操作

C-x b 切换文件
C-x 1 关闭其他窗口
C-x 2/C-x 3 打开其他窗口
C-x o 调到另一个窗口

1.7 其他

C-/ 撤销
M-/ 智能补全
M-$ 拼写检查
M-x 输入命令
C-g 取消命令

2 LaTeX编辑常见操作

2.1 自动匹配括号，$，上下标（+shift）

2.2 数学符号输入（希腊字母，运算符等），可自定义

` + 其他按键

2.3 修饰符字体/，可自定义

` （引号）

2.4 常见命令，可自定义（C-C ？ 查看）

fr TAB = \frac{}{}
sq TAB = \sqrt{}
equ* TAB = begin{equation*} \end{equation*}
qq TAB = \quad
ref TAB =\ref{...}
cte TAB =\cite{...}
sn TAB = \section{}
ss TAB = \subsection{}

2.5 环境

C-C C-e 生成新环境
C-u C-C C-e 更改环境
C-C ] 补全\end{...}

2.6 选择、注释与编译

C-x * 选中章节
C-x . 选中
C-C C-C 编译全文件
C-C C-r 编译区域
C-C C-v 查看PDF
C-C；注释/取消注释区域

2.7 LaTeX命令

C-C Enter (\label,\ref,\newcommand等等）

2.8 outline mode

C-o 命令前缀 （可自定义，默认为C-C C-@）
C-o C-a 展开全部
C-o C-o 折叠其他
C-o C-t 折叠所有章节
C-o C-e 展开当前章节

2.9 emacs内预览

C-C C-p C-p 预览
C-C C-p C-r预览区域
C-C C-p C-C C-r 取消预览区域
