1: !process 0 0 列进程 加载记事本!process 0 0 notepad.exe

2: dt显示结构dt !_KPROCESS + Address [ntdll包含所有函数及数据结构]

3: r cr3 != 1||2,

4: .process -i [ProcessAddress]

5: s -u 0x00000000 L?0x7fffffff "fileName"

6: du + StartAddress

7: .formats[StartAddress] ;转化为32地址

8：.formats 0y+[二进制数据]; 转化二进制数据为十六进制

9： r cr3;cr4;

10: !dd@cr3 ;@cr3是取cr3地址数据

11：!dq +[StartAddress + 偏移地址] 取的是64进制

12：dd查询虚拟地址；dd!查询物理地址
